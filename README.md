# 项目简介 java313

本项目是一套基于前后端分离架构的鲜花商城及花店管理系统。以下为项目的详细说明。

## 技术栈

- **前端**：Vue3
- **后端**：Node.js
- **数据库**：MySQL 5.7/8.0
- **Node.js版本**：14/16/18

## 功能模块

### 用户功能

- 登录注册
- 查看鲜花详情和分类
- 购物车管理
- 地址管理
- 结算流程
- 查看订单
- 修改个人信息

### 管理员功能

- 鲜花管理
- 类型管理
- 订单管理
- 用户管理
- 反馈管理
- 个人中心管理

## 系统角色

- 用户
- 管理员

## 运行环境

- 前端：现代浏览器环境
- 后端：支持 Node.js 14/16/18 的服务器环境
- 数据库：MySQL 5.7 或 8.0

- 基于Vue3，使用最新前端技术，确保系统UI/UX现代化。
